## ADDED Requirements

### Requirement: 任务页项目文件浏览器入口

系统 SHALL 在任务页提供任务内项目文件浏览器入口，用于查看当前任务绑定项目工作目录下的完整文件结构。

#### Scenario: 从任务页进入项目文件浏览器

- **GIVEN** 用户正在查看一个已绑定项目的任务
- **WHEN** 用户点击任务页中的“文件”入口
- **THEN** 系统应打开该任务对应项目的文件浏览器
- **AND** 文件浏览器应默认作用于当前任务绑定项目的工作目录

### Requirement: 浏览当前项目工作目录文件树

系统 SHALL 支持浏览当前任务绑定项目工作目录下的目录树，并限制访问范围不超出该工作目录。

#### Scenario: 展开目录查看子节点

- **GIVEN** 用户已打开任务内项目文件浏览器
- **WHEN** 用户展开某个目录节点
- **THEN** 系统应返回该目录下的直接子目录与文件
- **AND** 所有返回路径都必须位于该任务绑定项目的工作目录内

#### Scenario: 访问越界路径被拒绝

- **GIVEN** 某次文件树或文件内容请求试图访问项目工作目录之外的路径
- **WHEN** 服务端收到请求
- **THEN** 系统应拒绝该请求
- **AND** 返回明确的越界访问错误

### Requirement: 预览文本文件内容

系统 SHALL 支持在任务页中预览当前项目工作目录内的文本文件内容。

#### Scenario: 打开文本文件预览内容

- **GIVEN** 用户在文件浏览器中选中了一个文本文件
- **WHEN** 系统加载该文件内容
- **THEN** 系统应在任务页中显示该文件的只读预览
- **AND** 界面应展示当前文件的相对路径

#### Scenario: 打开二进制文件显示不可预览提示

- **GIVEN** 用户在文件浏览器中选中了一个二进制文件
- **WHEN** 系统尝试预览该文件
- **THEN** 系统不应直接展示原始二进制内容
- **AND** 应显示“当前文件暂不支持预览”之类的只读提示

#### Scenario: 打开超大文件显示截断提示

- **GIVEN** 用户在文件浏览器中选中了一个超出预览上限的文本文件
- **WHEN** 系统返回文件内容
- **THEN** 系统可以只返回预览上限内的内容
- **AND** 应告知用户当前内容已被截断

### Requirement: 手机端文件浏览可用

系统 SHALL 在手机端提供可用的任务内项目文件浏览体验，不要求与桌面端完全一致，但必须保证目录树与文件预览均可访问。

#### Scenario: 手机端切换目录树和文件预览

- **GIVEN** 用户正在手机端查看任务页中的项目文件浏览器
- **WHEN** 用户切换目录树与文件预览区域
- **THEN** 用户应能访问目录树并打开文件
- **AND** 用户应能看到对应文件内容或不可预览提示
